🚀 ChainView - Sistema de Gestão de Processos Hierárquicos

Sistema completo para gerenciamento de áreas e processos com estrutura hierárquica ilimitada, desenvolvido com arquitetura moderna separando frontend e backend.

Acesse a aplicação React com interface completa para gestão de processos

🎯 Funcionalidades Principais

  • Visualização Hierárquica: Árvore interativa de áreas e processos
  • Gestão de Processos: CRUD completo com subprocessos ilimitados
  • Visualização da Cadeia: Fluxogramas interativos com cores e ícones
  • API REST: Documentação Swagger completa
  • Interface Responsiva: Otimizada para desktop e mobile

🏗️ Tecnologias

  • Frontend: React + TypeScript + Ant Design + Vite
  • Backend: Laravel + PHP 8.2 + PostgreSQL
  • Infraestrutura: Docker + GitHub Actions + GitHub Pages
